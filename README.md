# Rupesh Jha Portfolio

Static portfolio website built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - page markup and content sections
- `css/styles.css` - CSS entry file
- `css/base.css` - reset, theme variables, base styles, animations
- `css/layout.css` - navigation, header, sections, footer layout
- `css/components.css` - cards, skills, projects, contact form, modal styles
- `css/responsive.css` - tablet and mobile responsive styles
- `js/project-data.js` - project detail content used in the modal
- `js/main.js` - interactions, animations, menu, modal, and contact form handling
- `images/` - profile and portfolio images
